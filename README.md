# Clothing-shop-Analyst

• فرایند شروع پروژه

داده ها با استفاده از python مرتب، اصلاح و دسته بندی شدند.
برخی داده های خام یا null در صورت نیاز جایگزین و برخی حذف شدند.
دیتا آماده ی وزود به فاز تحلیلی شد.

• شروع تحلیل و پیدا کردن بینش

بعد از کمی بررسی، ارتباطات میان جداول و ستون ها برقرار و ایجاد شد.
میزان مرجوعی ها نسبت به سال قبل ۲۱ درصد کاهش داشت اما همزمان فروش هم ۱۹ درصد نسبت به سال قبل کاهش داشت، یعنی کاهش مرجوعی ها، نه به خاطر اصلاحات بلکه نتیجه ی کاهش فروش بوده است.
فروش و سوددهی نیز نسبت به سال قبل کاهش داشت و نتوانست تارگت مورد نظر(شال قبل) را تاچ کند.

• راهکارهای مناسب 

جهت کاهش مرجوعی ها، پس از بررسی دیتای مورد نظر، راهکارهای زیر ارائه شد:
۱- آموزش به کارکنان برای تعامل بهتر با مشتریان و کسب رضایت بیشتر
۲-استفاده از کالاهای با کیفیت تر
۳-تصحیح قوانین و شفافیت بیشتر در پذیرش کالاهای مرجوعی همزمان با کسب رضایت مشتریان
۴-استفاده نکردن از رنگ های کم فروش در لباس ها مثل رنگ صورتی (طبق آمار مرجوعی ها)

و همینطور راهکارهای زیر جهت پیشبرد اهداف فروش ارائه شد:
۱-استفاده از طرح های تشویقی برای مشتریان وفادار و ترغیب بیشتر آنها به خرید
۲-کاهش حاشیه سود و افزایش مزیت رقابتی
۳-تغییر ساعات کاری شعب فروشگاه برای جوابگویی بهتر به مشتریان و جذب مشتریان هدف جدید
